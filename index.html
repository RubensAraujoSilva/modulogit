Módulo do Git

Configurando nossa máquina

Essa configuração é realizada globalmente para setar as informações na máquina tais como Nome e Email do desenvolvedor.

Definindo nome de usuário:
git config --global user.name "Rubens Araújo"
Definindo e-mail do usuário:
git config --global user.email "rubens@octosoft.com.br"
